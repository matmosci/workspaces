# Quick-start PlantUML boilerplate

## Initial setup

In ~/.gitconfig add an alias:

    [alias]
      puml = clone https://github.com/matmosci/puml.git uml

## Create new UML project

> git puml
> cd uml
> *rm -rf .git*
> sh run
